语句和表达式的区别在于，
语句主要为了进行某种操作，一般情况下不需要返回值； var a = 1 + 3; 这是一个语句
表达式则是为了得到返回值，`一定会返回一个值`。 1 + 3; 

语句以分号结尾，一个分号就表示一个语句结束。多个语句可以写在一行内。var a = 1 + 3 ; var b = 'abc';
表达式不需要分号结尾。一旦在表达式后面添加分号，则 JavaScript引擎就将表达式视为语句，这样会产生一些没有任何意义的语句。1 + 3;
